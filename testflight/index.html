Install Chat beta with TestFlight

Use TestFlight to install the Chat beta on your iPhone. First install the TestFlight app, then accept the public invite for the Chat beta build.

Download TestFlight Open Chat beta invite

How to install

  1. Tap Download TestFlight to install Apple TestFlight on your iPhone.
  2. Open the TestFlight app, then tap the public invite link above or paste it into Safari.
  3. Choose Accept and then tap Install for the Chat beta build.
  4. After install, open Chat from TestFlight and use it like a normal app. Updates will arrive through TestFlight.

Need help?

If the invite link does not open, make sure you are signed into the App Store with your Apple ID and installed TestFlight first. For support, email chat@phcreations.com.
